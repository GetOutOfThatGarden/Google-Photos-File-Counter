Google Photos File Counter

Google Photos File Counter is a command-line utility that allows users to analyze their Google Photos library by counting media files according to file type and extension.

The application requests read-only access to a user's Google Photos library solely for the purpose of analyzing and counting media items within the account.

No media content is modified, uploaded, or shared.

No personal data is stored on external servers.

All analysis is performed locally on the user's machine.

For more information about how user data is handled, please see the Privacy Policy.
