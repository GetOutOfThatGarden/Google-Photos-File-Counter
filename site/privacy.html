Privacy Policy

This application accesses Google Photos data only with the user's explicit consent through the Google OAuth 2.0 authentication flow.

The application requests read-only access to media metadata for the sole purpose of counting and categorizing files within the user's Google Photos library.

The application does not:

  • Modify or delete user media
  • Upload user data to any external server
  • Share user data with third parties
  • Store personal media content outside of the user's local environment

All processing is performed locally on the user's device.

Users may revoke access to their data at any time through their Google Account permissions.
